# Bridging-Coursework

This website using responsive frontend. The layout can be changed by window size. This website can displayed correctly on mobile devices or `1920Ã—1080` PC screen.

## Set up
First create a virual environment

Then install python dependencies:

`$ pip install -r requirements.tx`

Run local server:

`$ python manage.py runserver`

## Test 
Functional tests are in `functional_tests.py`

To run the test:
`$ python functional_tests.py`

Unit tests for cv are inside cv folder

To run the test:
`python manage.py test`
